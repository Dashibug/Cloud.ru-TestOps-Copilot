Блок 1. Начальная страница

- Отображается главная страница калькулятора с пояснительным текстом.
- Кнопка «Добавить сервис» отображается и кликабельна.
- Отображаются шаги процесса: «Добавление сервиса → Конфигурация → Подключение».
- Отображается текущая общая стоимость с НДС, единицы измерения «в месяц с НДС».
- Отображается дисклеймер об отсутствии публичной оферты.
- При первом заходе конфигурация пуста, стоимость равна 0 ₽.

Блок 2. Каталог продуктов

- Отображается каталог продуктов Evolution Cloud.ru.
- Отображаются категории продуктов: Инфраструктура, Сети, Хранение, Контейнеры, Платформы данных, ML/AI и др.
- Отображается список популярных продуктов.
- Для продуктов, где доступен Free Tier, отображается соответствующая пометка.
- Есть поиск по названию продукта в каталоге.
- Есть фильтрация по категориям и другим параметрам (например, по типу сервиса).
- При выборе продукта пользователь переходит к конфигурации этого продукта.

Блок 3. Конфигурация продукта (пример — Compute)

- Отображаются параметры конфигурации: количество vCPU, объём RAM, диск, регион, тариф.
- Для параметров используются элементы выбора (слайдеры, выпадающие списки), а не произвольный текстовый ввод.
- При изменении любого параметра конфигурации стоимость пересчитывается динамически.
- Для каждого параметра есть тултип с описанием (подсказка по наведению).
- При выборе региона корректно меняется стоимость (регион влияет на цену).
- Пользователь может указать количество экземпляров продукта; максимум 99 экземпляров.
- При недопустимом значении параметра показывается сообщение об ошибке или подсветка поля.

Блок 4. Управление конфигурацией

- Пользователь может добавить несколько сервисов в одну конфигурацию.
- Пользователь может удалить сервис из конфигурации.
- Доступен режим сравнения 2–3 конфигураций между собой.
- Пользователь может скачать расчёт конфигурации в формате PDF.
- Пользователь может скачать расчёт конфигурации в формате JSON.
- Пользователь может поделиться конфигурацией по ссылке (шаринг URL).
- Для каждого продукта есть кнопка «Подключить», кроме продукта Arenadata DB.
- Free Tier-конфигурацию одного и того же продукта можно добавить только один раз.

Блок 5. Mobile Responsiveness

- Интерфейс калькулятора корректно отображается на мобильных устройствах.
- Цена и ключевые элементы отображаются крупным шрифтом и читаемы.
- Все элементы управления (кнопки, селекты, поля ввода) доступны и кликабельны на мобильных.
- Горизонтальный и вертикальный скролл не ломают верстку карточек и таблиц.
- Каталог продуктов и экран конфигурации корректно открываются и закрываются на мобильных.
